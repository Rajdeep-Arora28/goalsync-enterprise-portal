# GoalSync Enterprise Portal

## ATOMQUEST Hackathon 1.0 Submission

GoalSync is an enterprise-grade Goal Setting & Tracking Portal designed to streamline employee performance management, quarterly check-ins, approval workflows, analytics, and governance tracking.

The platform provides a modern SaaS-style experience with role-based dashboards for Employees, Managers, and HR/Admin users.

# Core Validation Rules Implemented

- Total goal weightage must equal 100%
- Minimum weightage per goal = 10%
- Maximum goals per employee = 8
- Goals lock after approval
- Shared goals support
- Quarterly check-in windows
